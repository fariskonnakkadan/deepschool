# i DeepSchool Web

DeepSchool Web is a premium, AI-powered educational platform that leverages the **Gemini API** to dynamically generate learning modules for students from Class 1 to 12, as well as higher education courses.

## ‚ú® Key Features

- **AI Content Generation**: Powered by Google's Gemini models (Gemini 3 Pro, Flash, etc.) to provide detailed explanations, examples, and LaTeX-rendered math.
- **Premium Design System**: A sleek, glassmorphic UI with support for **Dark Mode** and **Light Mode**, plus a customizable accent color palette.
- **Persistent Knowledge**: Uses `localStorage` to save your API settings, browsing history, and content cache. Your learning environment stays exactly where you left it.
- **Intelligent Caching**: Content is saved locally after the first generation, making navigation instantaneous and reducing API consumption.
- **Global Search**: Instantly filter through dozens of generated subjects and topics.
- **PDF Export**: Generate clean, professional study notes with a dedicated print-to-PDF feature that hides UI elements like refresh buttons.

## üöÄ Getting Started

1. **Clone the repository** and navigate to the `/web` directory.
2. **Open `index.html`** in any modern web browser.
3. **Configure Settings**:
   - Click the **Gear icon** in the top right.
   - Enter your **Gemini API Key**.
   - Select your preferred model (e.g., `gemini-3-flash`).
4. **Start Learning**: Select a class and explore dynamically generated subjects and topics!
